# Bluedog
A sophisticated web application for discovering and analyzing blockchain wallet addresses with real-time data fetching and advanced risk assessment.
Features

• Multi-Blockchain Support

• Ethereum (ETH) - Full network support with real balance and transaction data
• Solana (SOL) - 4 Optimized for speed with parallel RPC endpoints
• Bitcoin (BTC) - Complete address analysis with transaction history

Core Features

• Real Blockchain Data - Fetches actual wallet balances and transaction counts
